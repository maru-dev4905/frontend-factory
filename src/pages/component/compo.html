Scroll Button

페이지에 하단이나 최상단 혹은 특정 위치로 이동할 수 있는 버튼입니다.
button 태그에 data-scr 속성을 추가하여 스크롤 방향을 지정할 수 있습니다.
data-scr 속성에 up을 입력하면 스크롤을 위로, down을 입력하면 스크롤을 아래로 이동합니다.
class에 scr_btn을 추가하여 스크롤 버튼을 사용할 수 있습니다.
특정 위치로 이동할 경우 data-scr 속성에 이동할 위치의 id 값을 입력합니다.

Modal

모달 팝업을 사용할 수 있는 컴포넌트입니다.
button 태그에 data-modal 속성을 추가하여 모달을 활성화할 수 있습니다.
data-modal 속성에 modal id 값을 입력하면 모달을 활성화합니다.
class에 wv_modal_btn을 추가하여 모달 버튼을 사용할 수 있습니다.

dialog 태그에 wv_modal 클래스를 추가하여 모달을 사용할 수 있습니다.
모달 내부에 닫기 버튼을 추가하려면 button 태그에 wv_modal_btn, close_modal 클래스와 data-modal 속성에 modal id 값을 입력합니다.

모달 내부에 닫기 버튼에 close_modal 버튼을 넣지 않고 data-modal 속성에 다른 modal id 값을 입력하면 현재 모달이 닫히고 다른 모달이 활성화 됩니다.

Tab

탭 컴포넌트입니다.
button 태그에 wv_tab_btn 클래스를 추가하여 탭 버튼을 사용할 수 있습니다.
div 태그에 wv_tab 클래스를 추가하여 탭 내용을 사용할 수 있습니다.
data-default 속성을 추가하여 기본으로 활성화할 탭을 지정할 수 있습니다. ( 0번부터 시작합니다 )
data-default 속성을 추가하지 않았을 때는 기본으로 첫 번째 탭이 활성화 됩니다.

탭 내부에 서브 탭을 사용할 경우 div 태그에 wv_tab 클래스를 추가하고 data-default 속성을 추가하여 기본으로 활성화할 서브 탭을 지정할 수 있습니다. ( 0번부터 시작합니다 )
